01

Objectif du projet

Dans le cadre du BTS SIO, la réalisation d'un portfolio numérique est exigée pour l'épreuve E5. Ce portfolio a pour objectif de présenter mes réalisations professionnelles, mes compétences techniques et mon parcours de formation, dans un format accessible en ligne à tout recruteur ou jury d'examen. J'ai choisi de développer ce site entièrement en HTML, CSS et JavaScript — l'intelligence artificielle Claude (Anthropic) a été utilisée comme outil de développement tout au long du projet.

02

Déroulement

1

Définition de l'arborescence du site : page d'accueil, présentation, BTS SIO SISR, veille technologique, mes projets classés par compétence du référentiel, épreuve E6.

ArchitectureRéférentiel BTS SIO
2

Choix d'une identité visuelle technique : thème sombre avec accents verts, police JetBrains Mono, terminal animé en page d'accueil, cohérente avec le profil SISR.

CSSDesignJetBrains Mono
3

Mise en place du système de navigation avec menu déroulant classant les projets selon les 6 blocs de compétences du référentiel BTS SIO. Protection par mot de passe des sections projets et E6.

JavaScriptNavigationSécurité
4

Hébergement du site sur GitHub Pages depuis un dépôt public (kassa0.github.io) avec versionning du code via Git et commits réguliers tout au long du développement.

GitHub PagesGitVersionning
5

Achat d'un nom de domaine personnalisé (portfolio.hprotteau.fr) chez OVH et configuration du CNAME pour le faire pointer vers GitHub Pages.

OVHDNSCNAME
03

Conclusion

🏁

Résultat obtenu

Portfolio en ligne accessible à l'adresse portfolio.hprotteau.fr, présentant l'ensemble des réalisations et compétences du BTS SIO SISR.

⚠️

Difficultés rencontrées

Organisation de l'arborescence et cohérence de la navigation entre les nombreuses pages. Configuration du nom de domaine personnalisé et des redirections DNS.

📈

Compétences acquises

Développement web front-end (HTML/CSS/JS), gestion de version Git/GitHub, déploiement d'un service en ligne, configuration DNS/CNAME.

Ce projet m'a permis de développer des compétences concrètes en développement web et en déploiement d'un service en ligne. Il illustre également ma capacité à utiliser des outils modernes comme l'IA pour gagner en efficacité tout en restant maître du résultat produit.
